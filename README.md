## 🚀 GitHub Pages 배포

`main` 브랜치에 변경사항을 올리면 GitHub Actions가 정적 페이지를 만들고 자동으로 배포합니다.

처음 한 번만 저장소의 **Settings → Pages → Build and deployment → Source**를 **GitHub Actions**로 설정해 주세요.

```bash
npm install
npm run dev
npm run build
```
